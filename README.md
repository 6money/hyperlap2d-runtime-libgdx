## HyperLap2D libGDX Runtime

HyperLap2D runtime for libGDX framework.

### Import

You can include through gradle

```
implementation '....'
```

### Support

You can learn how to use runtime in [Wiki](https://hyperlap2d.rednblack.games/wiki)
